# Slider AI

A web application that generates PowerPoint (PPTX) presentations using AI.

## Features

- **AI-Generated Slides**: Automatically creates slide content, titles, and speaker notes.
- **Image Integration**: Fetches relevant images from Unsplash.
- **Downloadable Output**: Generates `.pptx` files ready for download.

## Tech Stack

- **Backend**: Python (Flask)
- **Templating**: Jinja2
- **Presentation**: `python-pptx`

## Setup

1. **Clone the repository**

   ```bash
   git clone https://github.com/iamr7d/slider_ai.git
   cd slider_ai
   ```

2. **Install Dependencies**

   ```bash
   pip install -r requirements.txt
   ```

3. **Run the Application**
   ```bash
   python main.py
   ```
   Access the app at `http://localhost:5000`.
